# 02 - Control Flow

Once you have variables, you need to control what your program does with them.
This example covers the two main ways Java makes decisions.

## What this covers

- **if / else if / else** — runs a block of code based on a condition
- **switch** — cleaner alternative when checking one variable against fixed values
- **break** — stops execution from falling into the next case
- **default** — catches anything that doesn't match a case

## How to run

```bash
javac ControlFlowExample.java
java ControlFlowExample
```

## Expected output

```
=== If / Else ===
10 is positive.

=== Switch ===
Well done!
```
