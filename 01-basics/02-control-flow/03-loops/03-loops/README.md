# 03 - Loops

Loops let you repeat a block of code without writing it multiple times.
This example covers the three loop types Java provides.

## What this covers

- **for loop** — repeats a known number of times
- **while loop** — repeats until a condition becomes false
- **do-while loop** — like while, but always runs at least once

## How to run

```bash
javac LoopsExample.java
java LoopsExample
```

## Expected output

```
=== For Loop ===
Iteration: 0
Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4

=== While Loop ===
Iteration: 0
Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4

=== Do-While Loop ===
Iteration: 0
Iteration: 1
Iteration: 2
Iteration: 3
Iteration: 4
```
