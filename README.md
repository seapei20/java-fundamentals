# 01 - Basics

The starting point of any Java program. This example covers the core building
blocks you need before writing anything more complex.

## What this covers

- **Primitive types** — `int`, `double`, `boolean`, `char`
- **String** — why it is an Object, not a primitive, and how concatenation works
- **Arithmetic operators** — `+`, `-`, `*`, `/`, `%` including integer division
- **Comparison operators** — `>`, `==`, `!=` and why they return a boolean
- **Type casting** — widening (automatic) vs narrowing (explicit)

## How to run

```bash
javac BasicsExample.java
java BasicsExample
```

## Expected output

```
=== Primitive Types ===
Age: 21
Price: 9.99
Is on sale: true
Grade: A

=== String ===
Hello, Alice!
Name length: 5

=== Arithmetic Operators ===
a + b = 13
a - b = 7
a * b = 30
a / b = 3
a % b = 1

=== Comparison Operators ===
a > b: true
a == b: false
a != b: true

=== Type Casting ===
int 7 stored as double: 7.0
double 9.78 stored as int: 9
```
